# Playbook Fase 0: Coleta e Diagnóstico (Briefing)

**Objetivo:** Extrair o máximo de conhecimento do cliente para não começar a pesquisa do zero.
**Responsável:** Analista de Projetos / Comercial
**Entrada:** Lead qualificado ou Novo Cliente.

---

## 1. O Que Você Precisa Antes de Começar
Antes de falar com o cliente, certifique-se de ter em mãos (ou na tela):
- [ ] O arquivo `FLUXOGRAMA BUSCA POR PRODUTOS.md` (Seu roteiro mestre).
- [ ] O formulário `FORMULARIO_1_DADOS_EMPRESA.md` (Para preencher durante a call).
- [ ] Acesso ao site/catálogo do cliente.

## 2. Passo a Passo da Execução

### Passo 1: Qualificação Inicial (BANT)
*Faça estas perguntas mentalmente ou sutilmente para saber se vale a pena investir tempo.*

1.  **Budget (Orçamento):** "O cliente tem porte para pagar pela nossa consultoria/produtos?"
2.  **Authority (Autoridade):** "Estou falando com quem decide (Dono) ou quem influencia (Nutricionista)?"
3.  **Need (Necessidade):** "Eles têm uma dor latente (multa, quebra)?"
4.  **Timeline (Prazo):** "Para quando eles precisam resolver isso?"

### Passo 2: A Entrevista de Briefing (SPIN)
*Use o `FLUXOGRAMA BUSCA POR PRODUTOS.md` como guia, mas aplique a técnica SPIN para aprofundar.*

- **S (Situação):** "Me conte como é o processo de limpeza hoje. Quais produtos vocês usam?"
- **P (Problema):** "Quais são as maiores dificuldades? O cheiro incomoda? O piso mancha?"
- **I (Implicação):** "E quando o piso mancha, o que acontece? O cliente reclama? A vigilância aponta?" *(Aqui você descobre o valor do problema)*.
- **N (Necessidade):** "Se a gente conseguisse resolver essa mancha e ainda economizar produto, como isso ajudaria sua operação?"

### Passo 3: Coleta de Dados "Duros" (Obrigatório)
*Não encerre a reunião sem pedir estes itens. Eles são a munição para a Fase 1.*

1.  **Tabela de Produtos:** Excel ou PDF com tudo o que eles vendem/usam.
2.  **Fichas Técnicas / FISPQs:** Documentos técnicos dos produtos atuais (para comparativo).
3.  **Dados da Empresa:**
    - CNPJ (Para consultar CNAE e regime tributário).
    - Regime Tributário (Simples, Lucro Presumido, Lucro Real). **CRÍTICO** para argumentos fiscais.

### Passo 4: Organização Inicial
1.  Crie a pasta do cliente em `Arquivos com Argumentos/[Nome do Cliente]`.
2.  Salve a tabela de produtos e outros docs na subpasta `Arquivos sem tratamento`.
3.  Preencha o `FORMULARIO_1_DADOS_EMPRESA.md` com o que coletou.

---

## 3. Checklist de Validação da Fase 0
*Só avance para a Fase 1 se marcar TODOS os itens abaixo.*

- [ ] **Entrevista Realizada:** Conversei com o cliente e entendi o cenário básico.
- [ ] **Dores Mapeadas:** Identifiquei pelo menos 3 dores principais (ex: custo alto, risco de multa, ineficiência).
- [ ] **Produtos Listados:** Tenho a lista dos produtos que o cliente trabalha/usa.
- [ ] **Regime Tributário:** Sei se o cliente é Lucro Real, Presumido ou Simples (essencial para o argumento de PIS/COFINS).
- [ ] **Arquivos Salvos:** Todos os documentos recebidos estão na pasta do projeto.
