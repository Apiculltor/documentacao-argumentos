# Itens de Higiene Recomendados para Restaurantes: Sanitizantes, EPIs e Acessórios

## Sanitizantes (Produtos Químicos Sanitizantes)
Produtos de limpeza e desinfecção para serviços de alimentação, geralmente sem fragrância.

| Sanitizante | Indicações de Uso |
| :--- | :--- |
| **CIF Detergente Alcalino Clorado** | Detergente desinfetante à base de **cloro** para limpeza pesada. Indicado para pisos, bancadas e equipamentos em cozinhas. Limpa, clareia e desinfeta. |
| **CIF Desinfetante (Sem Perfume)** | Desinfetante sem fragrância para superfícies em contato com alimentos (utensílios, mesas, bancadas). Seguro para alumínio/inox. Alto rendimento. |
| **CIF Desinfetante Hortifrutícolas** | À base de **hipoclorito de sódio** para higienização de frutas, legumes e verduras. Aprovado pela ANVISA. |
| **CIF Limpador Peróxido** | Limpador concentrado à base de **peróxido de hidrogênio** para limpeza pesada e máxima desinfecção. Ideal para pisos, azulejos e sanitários. |
| **CIF Higienizador + Álcool** | Multiuso sem perfume com álcool para limpeza rápida por pulverização. Seca rapidamente. Ideal para mesas, balcões, caixas. |
| **OMO Desinfetante (Pisos Lavanda)** | **Perfumado**, para pisos e superfícies em áreas **não alimentícias** (salão, corredores, banheiros). |
| **Clorin Sanificante** | Sanitizante clorado de uso profissional. **Não requer enxágue**. Para desinfecção final de bancadas e pisos. |
| **Desinfetante Mirax-A** | Multiuso (marca alternativa) para pisos e superfícies em cozinhas. Aplicado com mop úmido. |
| **Chef-Clor** | Desinfetante clorado (marca alternativa) para cozinhas. Aplicado em limpeza corretiva de paredes, azulejos e pisos. |

## EPIs (Equipamentos de Proteção Individual)
É **obrigatório** usar EPIs antes de iniciar qualquer procedimento de higienização.

- **Avental Impermeável:** Protege contra respingos de químicos e gorduras.
- **Máscara de Proteção:** Reduz inalação de aerossóis químicos e previne contaminação.
- **Óculos de Proteção:** Protegem contra respingos de detergentes e soluções cloradas.
- **Touca (Proteção para Cabelos):** **Obrigatória** na cozinha e área de manipulação para manter a higiene.
- **Luvas de Látex (Cano Longo):** Protegem mãos e antebraços de químicos e sujeira. Recomenda-se separar por área (cozinha/banheiro) para evitar contaminação cruzada.
- **Calçados Antiderrapantes:** (Não listado no doc, mas recomendado) Essencial para prevenir quedas em pisos molhados.

## Acessórios de Limpeza e Utensílios

| Acessório | Aplicação / Indicação de Uso |
| :--- | :--- |
| **Rodo (Secador de Piso)** | Para remover excesso de água de pisos. Usar rodos específicos por área (cozinha/banheiro). |
| **Rodo Limpa-Vidros** | Squeegee de mão para limpeza de vidros e espelhos no salão e banheiros. |
| **Vassoura de Nylon** | Para varrer pisos e reunir resíduos sólidos. Usar cores diferentes por ambiente (amarela para cozinha, azul/vermelha para sanitários). |
| **Pá Coletora** | Para recolher a sujeira varrida. Usar pás exclusivas para cozinha e outra para salão/banheiro. |
| **Sistema Mop Úmido** | Conjunto de mop (esfregão) e balde com espremedor para lavagem úmida de pisos. Usar cores diferentes por área. |
| **Balde de Limpeza** | Para preparo de soluções. Ter baldes dedicados por área (amarelo para cozinha, vermelho para lixo/banheiro). |
| **Pulverizador Manual (Spray Bottle)** | Para aplicar produtos em superfícies menores (mesas, bancadas, vidros). |
| **Pulverizador Pressurizado** | Para aplicar solução sanitizante em áreas amplas (pisos, paredes) de forma rápida e homogênea. |
| **Escovas de Limpeza** | Cerdas firmes para esfregar superfícies. Usar escovas específicas por tarefa (grelhas, rejuntes, etc.) e setor. |
| **Suporte e Cabo Extensível** | Para alcançar e limpar locais altos (tetos, paredes, coifas). |
| **Fibras Abrasivas (Pads)** | Almofadas (tipo Scotch-Brite) para esfregar superfícies. Verde/amarela para gordura, branca para superfícies delicadas. |
| **Esponja Dupla Face** | Para lavagem manual de louças e utensílios. |
| **Panos de Limpeza (Multiuso e Microfibra)** | Pano multiuso para limpeza geral; pano de microfibra para acabamento e polimento (vidros, inox). |
| **Espanador Eletrostático** | Para remover pó de móveis e superfícies sem espalhar no ar. |
| **Placa de "Piso Molhado"** | Sinalização para prevenir acidentes enquanto o piso está secando. |
| **Lixeiras e Contentores de Lixo** | Com tampa e pedal na cozinha. Higienizar periodicamente. |
| **Raspadores e Rasquetas** | Para remover sujeiras incrustadas de chapas, fornos e pisos. |
